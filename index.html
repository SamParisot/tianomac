TianoMac Documentation
What is TianoMac?
TianoMac is an implementation of rEFInd. rEFInd is a useful bootloader for Mac
that is based on TianoCore (hence the name: TianoMac) that allows for booting
into UEFI drives that a Mac might not normally detect in its usual “Disk Select”
feature. TianoMac is a slightly more advanced version of rEFInd that allows it to
detect some more drives that even rEFInd can’t.
Why Should I Use TianoMac?
It is extremely useful for anyone wanting to run an operating system that Macs
usually can’t find such as Ubuntu.
What tools does TianoMac have?
TianoMac has a built in partition editor (gdisk) and Network Boot support (iPXE).
How do I Install TianoMac?
There are three ways to install TianoMac: External Install, Disc Install, and
Internal Install.
External and Disc Installs are great for temporary use and is the easiest to install.
However, if you want a more permanent installation, you’ll want to do an Internal
Installation.
External Install
This will require a flash drive. Any size will do because TianoMac is only 6.5MBs.
Back up any information you want to keep on it. Go to Disk Utility
(Finder>Applications>Utilities>Disk Utility) and click on your flash drive. Click
“Erase” and format it as “Mac OS Extended (Journaled”.
Now, download the TianoMac ZIP File. This contains the files needed to boot
TianoMac. That’s it! Now, shut down your Mac. Press the power button and hold
the Option/Alt key down. If you did this correct, you will see a menu that says
“Macintosh HD” and “EFI”. Click or select the one called “EFI” to boot TianoMac.
Disc Install
To install TianoMac to a CD or DVD, download the TianoMac ISO File. Insert a
blank CD or DVD. If a prompt comes up asking what you want to do with it, click
“Ignore”. Go to the folder containing the ISO (more than likely the Downloads
folder) and click the ISO file with two fingers (this is equivalent to a right-click).
Click on “Burn Disk Image “TianoMac-xx_disk.iso” to Disk” and click on “Burn” for
the next prompt. This will burn the image. When burned, TianoMac will be
installed on the disc. Shut down your Mac. Now, press the power button then
hold Option/Alt. Click the CD icon to boot into TianoMac.
Internal Install (ONLY DO THIS IF YOU HAVE DONE PARTITIONING
BEFORE)
Before we begin, don’t be scared to see your BootCamp partition
disappear for a little bit. This will be fixed after you boot into Windows
using TianoMac.
First, download the TianoMac ZIP File and extract it. We’ll come back to
this later.
Now, we get to the hard part - partitioning! Open Disk Utility and click
on your hard drive. Click “Partition”. Press the + button on the bottom
and change the size of the partition to 50MB. The partition table should

Choose “Mac OS Extended (Journaled)” as the format and click Apply.
When the operation is completed, copy over the TianoMac Files to the
new partition. That’s it!
Shut down your Mac. Press the power button and hold Option/Alt. Click
the Hard Drive icon labeled “EFI”. You should now boot into TianoMac!
For Those with BootCamp
Boot into your Windows partition using TianoMac. This will make your
Mac also recognize it again from the normal bootloader.
